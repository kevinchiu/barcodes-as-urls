Build Instructions
==================
Type "make" to create the Reader.app program.

Type "make clean" to delete all .o files as well as the application.

Usage
=====
Launch the application. Touch the screen to begin barcode recognition.

Troubleshooting
===============
To compile this source code, you must have iphone-dev toolchain version 
0.30 installed.

About
=====
This project is based on the Open Source QR Code Library, downloadable
from http://qrcode.sourceforge.jp/ .
